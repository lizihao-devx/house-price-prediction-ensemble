# 北京二手房（模拟）房价预估系统

## 简介

本项目基于`Scikit-learn`构建，通过约2.3万条北京二手房房价数据，利用随机森林算法实现对房产价值的自动化估值。

数据来源：[天池](https://tianchi.aliyun.com/dataset/206559)

## 方案

- 数据清洗：
  - 剔除错误/错位数据
  - 正则表达式提取和拆分户型
  - 根据日常经验逻辑补全缺失值
- 特征工程
  - 量化部分文本数据（户型、电梯、装修情况）
- 核心模型
  - Random Forest (100 Estimators)

## 关键结果：

*R²* = 0.64

主要预测因子：面积（62%）

## 快速开始

运行 python main.py 即可查看模型训练与模拟预测结果。
